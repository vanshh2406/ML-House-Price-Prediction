# House Price Prediction using Linear Regression

## Overview

This project predicts house prices using the Boston Housing Dataset and a Linear Regression model.

The objective is to understand the complete Machine Learning workflow including data preprocessing, feature selection, train-test splitting, model training, evaluation, and prediction.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn

## Dataset Features

* CRIM
* ZN
* INDUS
* CHAS
* NOX
* RM
* AGE
* DIS
* RAD
* TAX
* PTRATIO
* B
* LSTAT

Target Variable:

* MEDV (Median Value of Owner-Occupied Homes)

## Workflow

1. Load Dataset
2. Check Missing Values
3. Feature Selection
4. Train-Test Split
5. Train Linear Regression Model
6. Generate Predictions
7. Evaluate Performance
8. Visualize Results

## Model Performance

* R² Score: 0.6688
* Mean Absolute Error (MAE): 3.1891
* Adjusted R²: 0.6198262368988514

## Results

The model explains approximately 66.9% of the variance in house prices and achieves an average prediction error of about 3.19 units.
